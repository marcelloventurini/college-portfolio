# College Portfolio

Projeto desenvolvido como trabalho de faculdade do curso de Ciência da Computação para a disciplina de Desenvolvimento Web.

## Sobre o projeto

Este site foi criado como um portfólio simples e objetivo, com foco em clareza visual e navegação rápida. A página organiza as informações principais em quatro seções: apresentação, projetos, formação e contato.

## Funcionalidades

- Navegação interna com rolagem suave entre as seções.
- Header fixo que some ao descer a página e reaparece ao subir.
- AnimaçÕes de entrada dos blocos conforme o scroll.
- Layout responsivo para desktop e dispositivos moveis.
- Links para GitHub, LinkedIn, WhatsApp e e-mail.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como executar

1. Abra a pasta do projeto no VS Code.
2. Abra o arquivo [src/index.html](src/index.html) em um navegador.
3. Se preferir, use uma extensão como Live Server para recarregar a pagina automaticamente.
